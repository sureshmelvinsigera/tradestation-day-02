## Slides 27–29 | Summarizing Content – Context Windows | 1:50 – 1:55 (5 min)

- Slide 28: The chart shows context window sizes. Point out Claude Sonnet 4 and GPT-4.1 at the top (~1M tokens).
- **"The context window is the model's working memory. The bigger it is, the more of your document it can actually read and summarize in one shot — without losing the beginning by the time it reaches the end."**
- Slide 29 discussion: *"What problems might a very large context window create?"*
  - Good answers: attention dilution, slower processing, higher cost, privacy risk if sensitive data is included.
